knitcitations 0.0-1

While traditional Sweave documents can take advantage of LaTeX citation tools, knitr also allows users to combine the much simpler markdown with R code. This package tries offers citation tools to make up for this. This package will also take advantage of the web-based nature of markdown documents

  • authoryear_p: format the author and year parenthetically
  • authoryear_t: format the author and year
  • bibliography: Generate the bibliography
  • citep: Add a paranthetical citation
  • citet: Add a textual citation
  • cleanbib: clean the bib list
  • formatref: Convert crossref XML into a bibentry object
  • functiontable: Create a table of functions in a package
  • read.bibtex: read in bibtex and use key as list names
  • ref: Lookup article info via CrossRef with DOI.

Dependencies

  • Depends: knitr, bibtex
